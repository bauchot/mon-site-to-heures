  3. Pointer la caméra vers ce QR Code

🔄 Synchronisation COMPLÈTE (QR Code)

⚠️ Utiliser en cas de problème ou pour restaurer des données

⚠️ Attention :

  • Cette option remplace TOUTES les données du salarié
  • À utiliser pour :
    • Restaurer des données perdues
    • Synchroniser un nouveau téléphone
    • Résoudre des problèmes de données
